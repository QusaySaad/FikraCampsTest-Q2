### Write a program that accepts an array of numbers and returns an array of numbers in written form e.g.


[1,4,6] → [“one”, “four”, “six”]

[0,0,6,2,7] → [“zero”, “zero”, “six”,"two","seven"]

[5,4,3,2,1,5,8] → ["five", "four", "three", "two", "one", "five","eight"]
